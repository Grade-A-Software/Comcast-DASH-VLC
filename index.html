How to Run the Code

First, check out the source code for our version of VLC from git clone https://github.com/Grade-A-Software/Comcast-DASH-VLC.git

Then build (OSX),(Linux) the project and the plugin will be in the build directory.

Finally, from within the build directory, execute ./vlc --stream-filter comcast_dash

We are referencing this portion of the VLC codebase as a model for using DASH with VLC.

We have not identified why this does not work for Comcast
